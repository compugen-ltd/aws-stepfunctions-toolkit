# Product Overview

## Purpose
POC for using AWS Step Functions test API. 
for testing AWS Step Functions state machines locally using mock data

## Target Users
- AWS developers working with Step Functions
- DevOps engineers testing state machine workflows
- Teams implementing serverless orchestration patterns

## Key Features
- Local testing of Step Functions definitions without full deployment
- Mock state execution with predefined outputs
- Support for complex state machine flows including maps and tasks
- JSON-based configuration for test inputs and expected outputs
- Integration with AWS Step Functions test API for accurate simulation

## Business Objectives
- Reduce development cycle time by enabling local testing
- Improve reliability of Step Functions workflows through comprehensive testing
- Lower AWS costs by reducing need for live testing environments
- Enable test-driven development for serverless orchestration
