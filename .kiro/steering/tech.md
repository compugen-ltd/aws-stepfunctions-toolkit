# Technology Stack

## Core Technologies
- **Python 3.13+** - Primary development language
- **boto3** - AWS SDK for Python, Step Functions API integration
- **AWS Step Functions Test API** - Core testing functionality

## Development Environment
- **uv** - Python package manager and virtual environment
- **Jupyter/IPython** - Interactive development and analysis
- **Pycharm** - Primary IDE with Python extensions

## Data Handling
- **JSON** - Configuration files, state definitions, mock data
- **pandas** - Data analysis and manipulation
- **pathlib** - File system operations

## AWS Services
- **AWS Step Functions** - Target service for testing
- **AWS CLI** - Command-line interface for AWS operations

## Project Structure
- Virtual environment managed by uv
- JSON-based configuration for test cases
- Modular Python scripts for different testing scenarios
